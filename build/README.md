# Build Assets

Place your application icons here:

## Required Files

### macOS

- `icon.icns` - macOS app icon (1024x1024)

### Windows

- `icon.ico` - Windows app icon (256x256)

### Linux

- `icons/` directory with PNG files:
  - `16x16.png`
  - `32x32.png`
  - `48x48.png`
  - `64x64.png`
  - `128x128.png`
  - `256x256.png`
  - `512x512.png`

## Generating Icons

You can use tools like:

- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)
- [icon-gen](https://www.npmjs.com/package/icon-gen)
- Online converters like [CloudConvert](https://cloudconvert.com/)

Start with a 1024x1024 PNG and generate all required sizes.
